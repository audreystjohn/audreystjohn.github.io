User guide

  • Hold the "p" key down while clicking a vertex to pin it; do the same to unpin.
  • Hold the "d" key down while clicking a vertex to delete it.
  • Add a new vertex by clicking in blank space.
  • Add a bar by:
    1. Hold SHIFT key while clicking the first vertex; do the same to unselect.
    2. Hold SHIFT key while clicking a second vertex to add the constraint.
